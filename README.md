# 小信子语义理解api例程(已支持语音输入, 还没把播放的源码移过来)

功能：人机对话，智能助手。


操作方法：

    1, 编译好的程序在 App目录
    
    2, 运行-》通过麦克风输入语音
  
注意: 

    1, 把Libs/mips中讯飞库放到widora  /lib目录
    
    2, 如果要支持同时播放录音, 把SoundCardDriver下内核模块放到widora   /lib/modules/3.18.29/目录
